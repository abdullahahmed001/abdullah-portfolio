# Abdullah Ahmed Portfolio

Personal portfolio website for Abdullah Ahmed, a software engineering student at San Jose State University with interests in AI, DevOps, and full-stack engineering.

## Stack

- React
- Vite
- Plain CSS

## Run

```bash
cd /Users/abdullahahmed/abdullah-portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To GitHub Pages

This project includes a GitHub Pages workflow at `.github/workflows/deploy.yml`.

1. Push the project to a GitHub repository.
2. Set the default branch to `main`.
3. In GitHub, open `Settings -> Pages`.
4. Set the source to `GitHub Actions`.
5. Push to `main` and the workflow will publish the `dist/` output automatically.
